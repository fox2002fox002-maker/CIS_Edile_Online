
How to build the Release APK:

1. Make sure you have Android Studio and Android SDK installed.
2. Unzip this project folder.
3. Double-click on build-apk.bat
   - This will run Gradle and build the release APK automatically.
4. When it finishes, your APK will be here:
   app\build\outputs\apk\release\CIS_Edile_Online.apk
5. Copy the APK to your phone and install it.
